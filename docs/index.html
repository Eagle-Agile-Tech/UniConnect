UniConnect API (2.0.0)

Download OpenAPI specification:

REST API for UniConnect — university social & professional networking platform.

Features: • JWT + refresh token authentication • Role-based access (STUDENT, EXPERT, ADMIN, INSTITUTION) • Cursor-based pagination • Content moderation (automated + human review) • AI-powered recommendations (feed, users, communities) • Polymorphic reactions & media attachments • Communities, direct/group chats, networking (connections)
